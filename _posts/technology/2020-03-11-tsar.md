## gnuplot

- x轴是时间

例如：

```bash
"2013-07-22 15:59:00"   6286    3730
or
2013-07-22    6286    3730
```

对于第一种情况，设置如下（**注意大小写**）：

```txt
set timefmt '"%Y-%m-%d %H:%M:%S"'
set timefmt "\"%Y-%m-%d %H:%M:%S\""
```
