0x00 修改root密码

sudo adduser root
一般会提示root用户已经存在，需要使用下面的指令修改root用户密码：
sudo passwd root 
键入以上命令后，即可输入为root设置的密码。

0x06 安装zsh

还是习惯使用zsh。

  • 安装zsh
sudo apt install zsh
  • 修改默认的Terminal为zsh
chsh -s /bin/zsh
  • 安装oh-my-zsh
sh -c "$(wget https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh -O -)"

个人习惯，喜欢主题agnoster，需要修改.zshrc文件：

vim ~/.zshrc
//设置ZSH_THEME = "agnoster"

但是对于该主题，还需要安装相关字体：

mkdir ~/.fonts && cd ~/.fonts/ && git clone https://github.com/scotu/ubuntu-mono-powerline.git && cd ~

下载完成后，需要将下载的文件拷贝到:

/home/xxx/.local/share/fonts
//xxx为用户名

或者更加详细的步骤可查看：https://github.com/powerline/fonts，有详细的安装步骤。

0x07 安装docky

docky为一个类似MacOS系统的底部启动器软件程序，安装过程如下：

sudo add-apt-repository ppa:ricotz/docky
sudo apt-get update
sudo apt-get install docky
