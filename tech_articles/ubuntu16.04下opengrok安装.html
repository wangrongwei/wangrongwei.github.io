系统环境

  1. ubuntu16.04

安装步骤

  • 安装exuberant-ctags tomcat8

sudo apt-get install exuberant-ctags tomcat8
sudo /etc/init.d/tomcat8 restart

  • 测试是否安装成功
    在浏览器输入http://localhost:8080/看到:

  • 下载opengrok
    https://github.com/oracle/opengrok/releases/

下载后缀为.tar.gz即可。解压后，拷贝文件opengrok-x.xx.x/lib/source.war到/var/lib/tomcat8/webapps目录下。在浏览器输入：http://localhost:8080/source该文件将会自动生成source目录，但是到此步骤还没有指定源代码目录。

  • 指定代码目录

sudo ./OpenGrok index ~/xxx
xxx即代表需要阅读的源代码目录

  • 测试体验
    指定源代码时，代码过大，设置时间较长；如果该目录本身就是一个git工程，在opengrok上可直接看到每个版本的提交，这一点比较方便。
